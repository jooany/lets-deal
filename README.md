# lets-deal
가격제안을 통한 중고거래 플랫폼 서비스

### 사용 기술 및 개발 환경
Java, Spring Boot, IntelliJ, Gradle, Spring Data JPA, QueryDSL, MyBatis, PostgreSQL, AWS(EC2, S3, RDS,CodeDeploy), Redis
